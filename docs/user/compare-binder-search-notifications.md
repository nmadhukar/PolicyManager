# Compare, Evidence Binder, Search, And Notifications

This guide explains the document-management tools for reviewers, compliance staff, and clinic leaders.

## Compare Policy Versions

Open a policy and go to the version history. Use **Compare latest** or the row-level **Compare** action to compare an older version against the current version.

The comparison shows:

- added text
- removed text
- changed text
- metadata changes
- a summary count of changes

Reviewers can use this before approving or signing off on a policy. If extracted text is unavailable for either version, the screen explains that the comparison cannot be shown.

## Export A Compliance Evidence Binder

Open the policy sign-off page and use **Export binder**. Choose ZIP or combined PDF.

Binder sections can include:

- policy PDF with cover page
- cover page
- approval chain
- acknowledgment roster
- review history
- revision history
- audit log

The export is designed for CARF, JCAHO, internal audits, and annual review evidence. The original policy file is not changed by the export.

## Advanced Search

Use the document library filters to search by:

- title or document number
- policy text
- tags
- owner
- effective date
- next review date
- status
- extraction status

Quick filters help find policies that are expired, due soon, missing approval, or not acknowledged.

Use **Save search** to keep a private saved search. Select a saved search later to restore the saved filters.

## Notifications

Use the Notifications link in the main navigation or the badge in the top bar.

Notifications are created for:

- review assigned
- acknowledgment due
- policy published
- comment resolved
- approval requested

You can mark one notification as read, mark all as read, dismiss old notifications, and open linked policy work from the notification list.

Email digest preferences are managed on the Notifications page. Enable the digest, choose daily or weekly delivery, and set the preferred time and timezone.
