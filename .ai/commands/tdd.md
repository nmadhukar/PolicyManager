# Command: /tdd

## Purpose

Run a red-green-refactor loop for one focused behavior.

## Input

```text
/tdd <target>
```

## Procedure

1. Pick one acceptance criterion.
2. Write the failing test.
3. Run the focused test and confirm failure.
4. Implement the smallest change.
5. Run the focused test and confirm pass.
6. Refactor under green.
7. Record commands and result.
